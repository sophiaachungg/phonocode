How to use PhonoCode

Overview

PhonoCode is a scoring interface for reviewing phonological task responses. The model scores each audio response automatically — your job is to review low-confidence predictions and correct them before submitting.

Tasks

Phoneme Reversal Blending Nonwords NAART

Each task uses a separately trained model. Make sure you select the correct task before scoring.

Step-by-step

1
Enter your initials and select the task. The model will load — this may take a moment the first time.
2
Enter the participant ID (e.g. ReXa_090) and press Score →. The model will run inference on all audio files for that participant.
3
The table loads filtered to items flagged Needs Review — predictions where model confidence is below 75%. These are the ones that need your attention.
4
Click the ▶ play button on any row to listen to the audio file. Use the 0 / 1 toggle to set or correct the score.
5
Use Show all scores to see the full set of predictions and spot-check high-confidence items if needed.
6
Once all items are scored, press Submit scores →. Scores are written to your RA log and the task log.

Scoring guide

1 = correct response  ·  0 = incorrect response

If audio is missing, unclear, or unscoreable, leave it unscored and note it separately. Do not guess.

Notes

Your session stays active until you click Switch session. You can score multiple participants without reloading the model.

If the server is unreachable, confirm you are connected to the lab VPN.
